# Tables

This directory contains structured tables used in the LABRATE ROYALTY PRO methodology.

Tables are intended to ensure:
- transparency of calculations;
- reproducibility of results;
- linkage between legal objects and economic data.

Main categories of tables:
- correspondence between NICE classes and OKVED-2 activity codes;
- activity contours and comparable companies;
- normalized royalty rate distributions;
- auxiliary calculation matrices.
