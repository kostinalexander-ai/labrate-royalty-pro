# Case Studies

This directory contains case studies illustrating the application of the LABRATE ROYALTY PRO method.

Case studies are used to:
- demonstrate the role of activity contour definition;
- show the distinction between industry royalty invariants and calculated royalty rates;
- illustrate evidentiary logic in expert and judicial contexts.

Planned cases:
- Ricor Electronics (serial expert examinations and licensing analysis);
- royalty rate calculation for know-how in the space industry.
